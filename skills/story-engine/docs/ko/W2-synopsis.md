# W2: 시놉시스 + 프리플라이트

이 문서는 story-engine 스킬의 W2(시놉시스 작성 + 프리플라이트 체크) 단계 가이드입니다.

## 참조 문서 (해당 단계에서 반드시 읽을 것)

**장르별 메타프롬프트 (`{genre}` = `yadam` / `dark-history` / `bespoke`):**

| 단계 | 읽어야 할 문서 |
|------|--------------|
| 시놉시스 | `meta-prompts/{genre}/synopsis_guidelines.md` (yadam은 `야담_시놉시스_작성_지침.md`) |
| 프리플라이트 | `meta-prompts/{genre}/preflight.md` (yadam은 `야담_프리플라이트.md`) |

**Bespoke 장르 추가 필수 read**: `_story_source/_meta_supplement.md` (W1-5에서 생성된 이 에피소드 전용 보완) — universal base와 함께 read, **충돌 시 supplement 우선**.

## ★ 시놉시스 primary review lens — 몰입도

> SKILL.md 핵심 원칙: **궁금증 + 기대감 = 몰입도. 모든 wave의 최상위 평가 기준.**

W2 시놉시스는 다음 질문에 답해야 한다 (이 4개는 genre-agnostic 보편 기준 — `SKILL.md` 핵심 원칙에서 도출. bespoke 장르는 `meta-prompts/bespoke/preflight.md` § "★ Section 0 — Engagement (PRIMARY)"에 동일 기준이 5개 항목으로 expanded되어 있음. yadam / dark-history는 별도 Section 0 없이 본 lens 4개를 직접 적용):

- **매 챕터가 궁금증 유지 OR 기대감 누적 둘 중 하나에 기여하는가?** 둘 다 안 하는 챕터 = 구조 dead weight, cut or merge.
- **궁금증 / 기대감 둘 다 ch.15까지 살아있는가?** 한쪽만 작동하면 시청자 이탈.
- **ch.16 reveal이 궁금증을 닫고 새 기대감을 여는가?** 단순 closure는 약함.
- **첫 60초가 궁금증 promise를 거는가?** 분위기/배경만 깔면 클릭 안 됨.

20챕터 framework 자체는 도구일 뿐, 검토는 몰입도 곡선으로 한다.

## 시놉시스 작성

**`야담_시놉시스_작성_지침.md`를 반드시 읽고 시작한다.**

20챕터 프레임워크에 따라 시놉시스를 작성한다.

### 20챕터 구조

```
【기】 도입부 (1~5)
├─ 1. 시대적 배경 설정
├─ 2. 주인공 소개
├─ 3. 주변 인물 관계망
├─ 4. 평화로운 일상과 변화의 조짐
└─ 5. 사건 발생 — 일상의 균열

【승】 전개부 (6~12)
├─ 6. 첫 번째 선택과 행동
├─ 7. 이야기 흐름을 바꾸는 존재 등장
├─ 8. 첫 번째 시련과 대응
├─ 9. 첫 번째 반전 — 상황의 변화
├─ 10. 관계의 심화와 변화
├─ 11. 두 번째 시련 — 위기의 심화
└─ 12. 중요한 비밀/정보 발견

【전】 위기부 (13~17)
├─ 13. 주인공의 내적 성장과 변화
├─ 14. 핵심 반전 — 상황 역전
├─ 15. 최대 위기 — 절망의 순간
├─ 16. 결정적 선택과 행동
└─ 17. 절정 — 최종 대결/시험

【결】 결말부 (18~20)
├─ 18. 모든 것의 진실 드러남
├─ 19. 갈등 해소와 새로운 질서
└─ 20. 교훈과 여운
```

### 시놉시스에서 미리 설계할 것

- **궁금증 유지 설계 (최우선)**: 시놉시스 단계에서 "시청자가 범인/진실을 확신하는 최초 지점"을 명시하고, 그것이 15챕터(전 중반) 이후인지 반드시 확인
- **감정-행동 매핑**: 주요 감정 장면마다 Show 방법 기획
- **서스펜스 기법 배치**: 어느 챕터에 어떤 기법을 쓸지 설계
  - 기(1~5): 오픈 루프, 정보 격차, 시드 배치
  - 승(6~12): 극적 아이러니, 점진적 해빙, 에스컬레이션
  - 전(13~17): 클리프행어, 타임 프레셔, 거짓 해결, 패턴 파괴
  - 결(18~20): 콜백, 아이러니 역전, 작은 은혜의 복리
- **복선 추적표 초안**: 설치/회수 챕터 미리 설정
- **의심 대상 배치**: 진범 외 최소 2명의 미끼 용의자 설계
- **반전 갭 설계**: 진범을 시청자가 가장 신뢰하는 인물로 위장 → 폭로 시 갭 극대화. 반전 전까지 신뢰를 쌓는 장면을 충분히 배치
- **시네마틱 포인트**: 슬로모션, 병렬 편집 등 핵심 장면 설계

**출력 파일**: `04_시놉시스.md`

**리뷰 (서브스텝 2-1)** — 서브에이전트 자가검토 → 이슈 목록 → 수정. 최대 5회. 0 이슈 시 다음 서브스텝(2-2)으로 즉시 진행. 5회 초과 시 사용자에게 에스컬레이션.

## 프리플라이트 체크

**`야담_프리플라이트.md`를 반드시 읽고 시놉시스를 점검한다.**

### 절대 금지 확인

| # | 금지 사항 |
|---|----------|
| 1 | 감정 직접 명명 ("슬펐습니다", "기뻤습니다") |
| 2 | 말줄임표 (.., …) |
| 3 | 외래어 (키스, 다크서클 등) |
| 4 | 영어/알파벳 일체 |
| 5 | 아라비아 숫자 (8시 → 여덟 시) |
| 6 | '그', '그녀' 지칭 → 이름/호칭 사용 |
| 7 | "~씨" 호칭 → 낭자, 도령, 마님 등 |
| 8 | "하녀" → 몸종, 침모 |

### 시놉시스 구조 점검

- [ ] 의심 대상 최소 2명 배치되었는가?
- [ ] 반전이 2중 이상 설계되었는가?
- [ ] 궁금증이 전 중반(15챕터)까지 유지되는가? → 시청자가 범인/진실을 확신하는 최초 지점이 15챕터 이후인지 확인
- [ ] 서스펜스 기법이 각 구간에 1개 이상 배치되었는가?
- [ ] 복선 추적표가 완성되었는가?
- [ ] 감정선 곡선이 올라갔다 내려갔다 교차하는가?

### 복선 추적표 양식

|---|----------|----------|----------|------|
| 1 | | | | 미설치/설치완료/회수완료 |

**출력 파일**: `05_프리플라이트.md`

## Wave 리뷰 요약
위 각 서브스텝은 최대 5회 리뷰(0 이슈 시 즉시 진행)를 강제한다. 마지막 서브스텝의 리뷰가 통과하면 Wave 2 완료. 어느 서브스텝이든 5회 초과 시 사용자에게 에스컬레이션.
